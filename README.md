# Study Launcher (Android 16)

This Android app schedules each provided lesson as an exact weekly alarm. At alarm time it tries to open the YouTube app (`com.google.android.youtube`) with the matching video. If YouTube is absent, it opens the same link in the default browser.

## Device setup

1. Install the built app.
2. Open **Study Launcher** and choose **Enable automatic schedule**.
3. In Android's settings screen, allow alarms and reminders.
4. Keep YouTube installed for direct YouTube launches.

Android 16 may prevent any app from placing another app on screen in some security states (for example, an actively locked device). The app still schedules all lessons and attempts the direct launch as requested.
